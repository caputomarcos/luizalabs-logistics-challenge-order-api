Carregar e processar múltiplos arquivos

Permite o carregamento de múltiplos arquivos, processando-os e retornando os resultados agregados.

Request Body schema: multipart/form-data
file
Array of file <binary> [ items <binary > ]

Responses

production server

http://localhost:6868/api/upload

Obter pedidos da memória

Retorna a lista de pedidos armazenados na memória com base nos filtros fornecidos.

query Parameters
userId
number

ID do usuário para filtrar os pedidos.

orderId
number

ID do pedido para filtrar os pedidos.

startDate
string

Data de início para filtrar os pedidos (formato YYYY-MM-DD).

endDate
string

Data de fim para filtrar os pedidos (formato YYYY-MM-DD).

Responses

production server

http://localhost:6868/api/memory/orders

orderId
